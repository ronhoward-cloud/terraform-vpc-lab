\# Terraform VPC Lab (Public + Private Subnets)



This lab provisions a basic AWS network using Terraform.



\## What it builds

\- VPC

\- Public subnet

\- Private subnet

\- Internet Gateway

\- Route tables and associations

\- Security group

\- EC2 instance (validation)

\- Clean teardown with `terraform destroy`



\## How to run

terraform init

terraform validate

terraform plan

terraform apply



\## Teardown

terraform destroy
